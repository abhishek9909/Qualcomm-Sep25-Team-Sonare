# Qualcomm-Sep25-Team-Sonare
GitHub repo for team sonare - Qualcomm Hackathon Sep 2025


### uv setup

1. Install uv as:
```bash
brew install uv 
```

2. Install pyenv as:
```bash
brew install pyenv
https://github.com/pyenv/pyenv
```

3. Setup uv venv
```bash
uv venv
source .venv/bin/activate 
```

4. Sync Dependencies
```bash
uv sync 
````

5. Add new dependencies
```bash
uv add --package-name 
```
